# AutoDeploy.AI Landing Page

This is a responsive landing page for AutoDeploy.AI, built with HTML, CSS, and JavaScript.

## Features
- Responsive design
- Modern dark UI styling
- Interactive JavaScript elements

## Setup
1. Clone the repository.
2. Open `index.html` in your web browser.

## Usage
- Click the "Learn More" button to scroll to the About section.
- Fill out the contact form and submit to see a confirmation message.

## Deployment
This project is a static website and can be deployed on any static hosting service like GitHub Pages, Netlify, or Vercel.
